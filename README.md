# ⚡ ControleFirme — Seu Aliado Financeiro Inteligente

O **ControleFirme** é uma aplicação web moderna de educação e planejamento financeiro projetada especificamente para freelancers, autônomos e profissionais da *gig economy*. Diante da oscilação natural de renda que caracteriza o trabalho independente, o ecossistema foi estruturado para calcular colchões de segurança ideais, prever cenários com base em simulações de dados inspiradas no ecossistema Open Finance e fornecer clareza mental e estabilidade na gestão do dinheiro.

Visite a aplicação publicada: [https://controle-firme-react.vercel.app/](https://controle-firme-react.vercel.app/)

🛠️ Tecnologias Utilizadas
​O projeto foi construído empregando as melhores práticas de desenvolvimento front-end moderno, unindo robustez lógica e uma experiência de usuário rica:
​React & TypeScript: Arquitetura componentizada para reatividade fluida e tipagem estática rigorosa para mitigar falhas em tempo de execução.
​Tailwind CSS: Abordagem utilitária de estilização, proporcionando uma interface responsiva, com paletas escuras elegantes (bg-slate-900) e elementos altamente polidos.
​Vite: Ferramenta de build de última geração que viabiliza um ambiente de desenvolvimento ultra veloz.
​Termux: Utilizado como ambiente de desenvolvimento móvel baseado em terminal para o controle de versão e operações de sistema.
​Vercel: Infraestrutura de nuvem global configurada para Continuous Deployment integrado diretamente ao repositório git.
​🚀 Funcionalidades Principais
​Simulação Open Finance: Experiência imersiva que simula a integração de contas bancárias para coletar dados históricos de receitas.
​Buffer de Oscilação de Renda: Cálculos algorítmicos automáticos que determinam o tamanho ideal da reserva financeira (colchão de segurança) considerando os meses de vacas magras.
​Interface Centrada em UX: Visual minimalista focado na redução da ansiedade financeira e no aumento do controle psicológico do usuário sobre o patrimônio.
​Hospedagem Automatizada: Ciclo completo de entrega com builds automáticos ativados a cada envio ao repositório oficial.
